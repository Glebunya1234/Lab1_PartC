# Модифіковані жорданові виключення (МЖВ) для мішаних систем ЗЛП

## Опис

Цей проєкт є вдосконаленою версією вебзастосунку для розв'язання задач лінійного програмування (ЗЛП) симплекс-методом із застосуванням **модифікованих жорданових виключень (МЖВ)**. Головною особливістю оновлення є підтримка мішаних систем обмежень, що включають як нерівності ($\le, \ge$), так і рівняння ($=$).

Проєкт забезпечує повну автоматизацію процесу:

- **Попередня обробка:** Трансформація мішаних обмежень у канонічний вигляд з автоматичним інвертуванням знаків для забезпечення коректності вільних членів.
- **Етап 0 (Нуль-рядки):** Видалення штучних нуль-рядків, що відповідають обмеженням-рівнянням, для приведення системи до базисного вигляду.
- **Етап 1 (Опорний розв'язок):** Пошук початкового допустимого плану.
- **Етап 2 (Оптимальний розв'язок):** Знаходження екстремуму (Max або Min) цільової функції.
- **Детальне логування:** Генерація академічного протоколу обчислень із виведенням розв'язувальних елементів та форматованих симплекс-таблиць на кожному кроці.

---

## Використані технології

- **HTML5/CSS3:** Адаптивний інтерфейс для введення даних.
- **JavaScript (ES6):** Матричні обчислення, алгоритм МЖВ та динамічне формування протоколу.

---

## Запуск

1. Склонуйте репозиторій або завантажте файли проєкту.
2. Відкрийте файл `index.html` у будь-якому сучасному веббраузері.
3. Вкажіть кількість змінних, цільову функцію та введіть систему обмежень (рівняння та нерівності у кожному рядку).
4. Натисніть кнопку **"Розв'язати"**, щоб отримати покроковий аналіз та фінальний оптимальний план $X$.
