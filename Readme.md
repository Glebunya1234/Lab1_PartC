# Модифіковані жорданові виключення (МЖВ) для задач ЛП

## Опис

Цей проєкт реалізує алгоритм **модифікованих жорданових виключень (МЖВ)** для розв'язання задач лінійного програмування (ЗЛП) симплекс-методом. Програма дозволяє знайти оптимальний план для цільової функції за заданої системи обмежень-нерівностей.

Проєкт автоматизує:

- Перетворення вихідних нерівностей у канонічний вигляд для МЖВ.
- Пошук початкового опорного розв'язку (Етап 1).
- Пошук оптимального розв'язку (Етап 2).
- Формування детального покрокового протоколу обчислень у вигляді симплекс-таблиць.

---

## Використані технології

- **HTML5/CSS3:** Структура та оформлення інтерфейсу.
- **JavaScript (ES6):** Логіка парсингу математичних виразів та матричних обчислень.

---

## Запуск

1. Склонуйте репозиторій або завантажте файли проєкту.
2. Відкрийте файл `index.html` у будь-якому сучасному браузері.
3. Введіть кількість змінних, цільову функцію та систему обмежень.
4. Натисніть кнопку **"Розв'язати"** для отримання повного протоколу та результату.
